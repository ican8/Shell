# Shell
Proof of Concept project for Operating System shell. Simple command interpreter written in C

# Features :-
-   Read commands in a loop,
-   show prompt,
-   run commands,
-   run commands in background,
-   \> (output redirection),
-   < (input redirection) and
-   | (pipe)
-   Handle signals Ctrl-C and Ctrl-z
-   fg , bg
